### Google Drive File Downloader CLI 

#### Installation guide
    1. cd ~/
    2. $git clone
    3. $chmod 777 ./install.sh
    4. $sudo ./install.sh

#### Guid to uninstall 

    1. $sudo /usr/local/bin/uninstall_gdrive_dl

#### Usage
    
    $gdrive_dl <file_id> <file_name>
    
    $gdrive_dl 1LmY-hgYBPbhLk0aovxd2c33IHF1KY_nZ  sample.jpeg 

#### Platform supported
    1. Mac OS
    2. Linux
